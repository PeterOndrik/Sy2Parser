typedef void SY2PARSER_API_CALL ParsedNodeCallback(Sy2ParserHandle handle, const T_Sy2Node *node, void *callbackContext)

Callback function which is called after a node is completely parsed.

Otherwise the ParsingErrorCallback is called.

The entire subtree must be read by the sy2ReadNext() function. For example:

1 void SY2PARSER_API_CALL parsedNodeCallback(Sy2ParserHandle handle, const T_Sy2Node *node, void *callbackContext)
2 {
3  // the first node
4  printf("%u: %s\n", node->type, node->value);
5  T_Sy2Node newNode = { 0 };
6 
7  // read first child node
8  Sy2ParserStatus status = sy2ReadNext(handle, &newNode);
9  while (status == SY2_SUCCESS && newNode.depth > node->depth)
10  {
11  // display child node
12  printf("%u: %s\n", newNode.type, newNode.value);
13  // read next child node
14  status = sy2ReadNext(handle, &newNode);
15  }
16 }
Parameters
[in]handleA descriptor identifying an open Sy2 parser.
[in]nodeInformation about parsed node.
[in]callbackContextThe pointer value that was passed when the callback was registered.

Returns the API version implemented by the DLL.

Returns
This function returns the version number of the programming interface (API) implemented by the DLL. The number returned is the SY2PARSER_API_VERSION constant the DLL has been compiled with. For example 1.2 = 0x00010002.

Open a Sy2 Parser by using a file name.

Parameters
[in]fileNameSpecifies the name of a file to be parse.
[out]handleAddress of a caller-provided variable which will be set to a handle value if the function succeeds. The handle represents a Sy2 Parser instance and is required for all subsequent API calls. Note that the value SY2PARSER_INVALID_HANDLE (zero) is an invalid handle. If the function succeeds it will return a valid non-zero handle.
Returns
The function returns SY2_SUCCESS if successful, an error code otherwise. Some specific error codes are described below.
Return values
SY2_FILE_NOT_FOUNDThe file is not present.
SY2PARSER_API Sy2ParserStatus SY2PARSER_API_CALL sy2Close ( Sy2ParserHandle  handle)

Close an opened Sy2 Parser.

The specified Sy2 Parser will be closed and the provided handle becomes invalid after this call. The handle must not be used with any further API calls.

Parameters
[in]handleA descriptor identifying a Sy2 Parser. The specified handle becomes invalid after this call and must not be used with any subsequent call.
Returns
The function returns SY2_SUCCESS if successful, an error code otherwise. If a valid handle is specified, the function returns always SY2_SUCCESS.

SY2PARSER_API Sy2ParserStatus SY2PARSER_API_CALL sy2Parse ( Sy2ParserHandle  handle)

Perform a single-call parse of an Sy2 file associated with a handle. This function internally creates AST which can be read by using sy2ReadNext() funcition after or during parsing.

While sy2Parse() is running the previously registered callbacks are invoked. Functions sy2SetParsingErrorCallback(), sy2SetParsingProgressCallback() and sy2AddParsedNodeCallback().

Parameters
[in]handleA descriptor identifying an open Sy2 parser.
Returns
The function returns SY2_SUCCESS if successful, an error code otherwise.
Return values
SY2_PARSING_ABORTEDThe parsing has been canceled. The AST may be incomplete.
